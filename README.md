# 集体记忆社会-水文模型
这个模型来自于对前人模型的实现和加工，通过将集体记忆区分为文化记忆和交流技艺，
它的主要目的是强调集体记忆在社会-水文系统中所起的重要作用。
我们指出利用区分集体记忆模型可以对如何应对洪水提供新的见解。
